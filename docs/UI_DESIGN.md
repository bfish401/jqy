# JQY 用户界面设计规范

## 设计原则

### 1. 简洁易用
- 遵循极简设计原则，减少不必要的元素
- 提供清晰的视觉层级
- 突出核心功能和最常用操作

### 2. 一致性
- 统一的配色方案
- 一致的排版和间距
- 规范的组件使用

### 3. 响应式设计
- 支持桌面、平板、手机设备
- 自适应布局
- 触摸友好的交互

### 4. 可访问性
- 支持键盘导航
- 足够的对比度
- 清晰的焦点指示

- 等宽字体: 'Courier New', monospace

### 中文
- 主字体: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, '微软雅黑', sans-serif

### 字号
- 页面标题 (H1): 32px / 400 weight
- 章节标题 (H2): 24px / 500 weight
- 小标题 (H3): 18px / 500 weight

- 标签/标记: 12px / 500 weight

### 认证方式标签页

- **标签样式**: 胶囊型按钮，宽度相等
- **激活状态**: 蓝色背景 (#1890FF)，白色文字
- **非激活状态**: 浅灰背景 (#F5F5F5)，黑色文字
- **悬停效果**: 背景色变暗，显示光标

### 表单输入框

- **高度**: 40px
- **边框**: 1px solid #D9D9D9
- **圆角**: 4px
- **焦点**: 边框变蓝 (#1890FF)，显示蓝色阴影
- **禁用**: 背景灰色，文字变淡
- **错误**: 边框变红 (#F5222D)，显示错误提示

## 主界面布局详细设计

### 顶部导航栏

```

背景: 白色 #FFFFFF (浅色模式) / #262626 (深色模式)
阴影: 0 2px 8px rgba(0, 0, 0, 0.1)

左侧:
  Logo 32x32px + 应用名称 (18px / 600 weight)
  距左边: 16px

中央:
  面包屑导航 (12px)
  示例: 首页 > 资产管理 > 工具列表

右侧 (距右边: 16px):
  1. 全局搜索框 (宽度: 200px，高度: 36px)
  2. 消息铃铛 (大小: 20px，显示红色徽章数字)
  3. 用户菜单 (头像 36x36px + 下拉箭头)
  4. 设置按钮 (仅管理员，大小: 20px)
  5. 主题切换 (月亮/太阳图标)
  
  各元素间距: 8px
```

## 动画规范

### 缓动函数
```
标准: cubic-bezier(0.4, 0, 0.2, 1)
快速: cubic-bezier(0.4, 0, 1, 1)
缓慢: cubic-bezier(0, 0, 0.2, 1)
```

### 动画时长
```
快速操作: 200ms (按钮、表单)
过渡动画: 300ms (路由切换、折叠)
复杂动画: 500ms (对话框、通知)
```

## 响应式断点

```
移动设备:     < 576px
平板设备:     576px - 992px
桌面设备:     > 992px

调整方案:
- 移动: 侧边栏隐藏，使用汉堡菜单
- 平板: 侧边栏可折叠
- 桌面: 侧边栏固定显示
```

## 深色模式

### 主要颜色调整
```
背景色:       #141414 (深色背景)
卡片背景:     #1F1F1F (卡片背景)
文字色:       #FFFFFF (主文字)
次文字:       #C9C9C9 (次要文字)
边框色:       #434343 (边框)
```

### 过渡效果
- 色值过渡时长: 200ms
- 使用 prefers-color-scheme CSS 媒体查询
- 支持手动切换，保存用户偏好到 localStorage

## 无障碍设计

### 颜色对比度
- 大文字: >= 3:1
- 普通文字: >= 4.5:1
- 图形元素: >= 3:1

### 焦点管理
- 清晰的焦点轮廓 (最小宽度 2px)
- Tab 键可访问所有交互元素
- 焦点顺序符合逻辑

### 标签和说明
- 表单输入必须有 label
- 按钮必须有明确的文字或 aria-label
- 错误消息清晰明确
